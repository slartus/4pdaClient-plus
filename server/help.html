Для использования Удаленного Сервера для заметок в приложении необходимо разместить на своём сервере файл notes.php. Ваш сервер должен поддержитьва php не ниже версии 5.6
Для заметок используется БД Sqlite со структурой аналогичной структуре в клиенте.
Если вы хотите перенести заметки с локального хранилища на Удалённый, то сделайте бэкап в клиенте, а затем полученный файл закачайте на сервер и положите рядом с notes.php с именем notes.sqlite

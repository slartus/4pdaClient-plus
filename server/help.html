Для использования Удаленного Сервера для заметок в приложении необходимо разместить на своём сервере файл notes.php.

Ваш сервер должен поддержить php не ниже версии 5.6

Для заметок используется БД Sqlite со структурой аналогичной структуре в клиенте. Файл БД с заметками называется notes.sqlite и будет создан автоматически

Если вы хотите перенести заметки с локального хранилища на Удалённый, то сделайте бэкап в клиенте, а затем полученный файл закачайте на сервер и положите рядом с notes.php с именем notes.sqlite

Отредактируйте файл notes.php и замените значение переменной в 3 строке $PASSWORD на свой пароль. например, $PASSWORD = "MY_SUPER_PASSWORD";

Укажите в клиенте путь до файла notes.php с параметром password=MY_SUPER_PASSWORD. например, http://myserver.ru/notes.php?password=MY_SUPER_PASSWORD

Клиент сделает запрос и проверит правильность пути
